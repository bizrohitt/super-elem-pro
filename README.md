# Super Elem Pro 🚀
Created by **Rohit** | *Personal Use Only*

Super Elem Pro is a high-performance, modular Elementor addon designed to unlock "Pro" capabilities using 100% open-source methods.

## ✨ Core Modules
- **Theme Builder**: Custom Headers, Footers, and Single Post templates.
- **Popup Builder**: Exit-intent and scroll-based triggers.
- **Advanced Forms**: Multi-step forms with database storage.
- **Dynamic Tags**: Pull data from ACF, Pods, or Meta Box.
- **Extensions**: Sticky elements, parallax, and custom CSS.

## 🛠 Installation
1. Download the latest `super-elem-pro.zip` from the [Releases](../../releases) tab.
2. Upload to your WordPress site via **Plugins > Add New**.
3. Activate the plugin.

## 👨‍💻 Development
1. Clone the repo: `git clone <your-repo-url>`
2. Install dependencies: `composer install`
3. Push to `main` to trigger the **Auto-Release** workflow.

## 🔒 Code Protection
The code is automatically obfuscated during the GitHub Release process to prevent unauthorized derivative works.
